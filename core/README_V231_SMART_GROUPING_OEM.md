# V231 Smart Grouping OEM

Cambios aplicados solo en `control-plan.html`:

- Bloques previos de OP20 quedan al inicio de OP20, antes de las características de producto.
- Preparación, puesta a punto, puesta cero de equipos y control volante se mantienen como subbloques internos de OP20.
- Texto completo visible: celdas con autoaltura, sin recortes ni dependencia de hover.
- Agrupación inteligente de cotas no KCS consecutivas cuando comparten método, instrumento, muestra, frecuencia, responsable y reacción.
- KCS permanecen separadas y visibles.
- Frase metrológica fija mantenida en cabecera.
