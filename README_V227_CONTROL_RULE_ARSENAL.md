# V227 CONTROL RULE ARSENAL

Motor de criterios reforzado a partir de planes de control reales aportados.

No se toca el motor dimensional, ISIR, KCS ni Dimensional. Solo se actualiza control-plan.html.

## Reglas nuevas principales

- KCS geométricas y dimensionales con frecuencia reforzada.
- Roscas con tampón P/NP.
- Rugosidad con rugosímetro.
- Posición/perfil/orientación con 3D.
- Diámetros H7/H6 y tolerancia estrecha con alexómetro/micrómetro 3 contactos.
- Profundidades con sonda/pie de rey/CMM.
- Radios con plantilla + presetting.
- Chaflanes y ángulos con presetting/pie de rey/CMM.
- Distancias entre planos con micrómetro/gramil/CMM según tolerancia.

## Fuente

Se han utilizado los planes de control históricos cargados en la conversación como base de criterios.
