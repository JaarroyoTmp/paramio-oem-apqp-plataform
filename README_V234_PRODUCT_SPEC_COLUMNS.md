# V234 PRODUCT / SPEC COLUMNS

Cambios solo en `control-plan.html`:

- Columna Producto = tipo de característica industrial: Posición, Perfil, Distancia, Diámetro, Radio, Rugosidad, Profundidad, Chaflán, etc.
- Columna Proceso queda vacía para cotas procedentes de Inspection.
- Especificación de producto/proceso/tolerancia contiene la cota real y sus límites/tolerancias.
- Cotas individuales, sin agrupación.
- No se toca ISIR, KCS, Dimensional ni motor dimensional.
