# Paramio OEM APQP Platform

Plataforma documental OEM/APQP para automatización de documentación de calidad industrial.

## Versión estable inicial

**V250 Full Translation**

## Módulos

- Inicio / carga documental
- Diagnóstico trazable
- ISIR
- Dimensional
- KCS
- Plan de Control
- Flowchart
- Traducción ES/EN
- Dimensional multicolumna
- Selector KCS visual

## Uso local

Abrir `index.html` en navegador.
